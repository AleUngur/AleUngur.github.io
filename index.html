Weather App JavaScript

With this app I coded, I learned about API and how can it be used to get information.

Weather App

This app is for knowing the weather in different locations. I used Open Weather to get the information from the server. I created a function 'getWeather' for downloading the information about the weather using the API, and the 'Get' method I coded using Fetch. Another function I created was 'getCities' for accessing the location I wanted the weather from, 'Get' was also the method used, but the information was given from a .js document with an array of a few cities. The temperature given from the server was in Fahrenheit degrees so I also made a function to transform those in Celsius degrees. For styling, I used Bootstrap elements.
Creating this web app I learned so much about HTTP, APIs, and the basics of Javascript. I learned to use: variables, constants, and operators, arrays, objects, functions/arrow functions, loops, events. I also got to know about DOM and the HTTP methods: get, post, put, delete.

About me

I'm Alexandra!

I am a 23 years old girl. This year (2021), I graduated from the Electronics, Telecommunications, and Information technology college from the Polytechnic University of Timisoara. Four full years of learning all about engineering and IT. What I was always focused on was Programming. I loved to write code and solve problems. I learned the basics of C++, C, Phyton, Java, and SQL. In my senior year, I got into creating websites and designing them. I created a personal site using HTML & CSS, Bootstrap, and a little JavaScript. I enjoyed doing it.

Where I'm from

I am from a little town not far from Timisoara, Romania. I grew up drawing, solving math problems, reading, and putting together puzzles. I always considered myself a creative person, a visualizer rather than an analytical person. When I first got into Informatics and coding in high school, I was thrilled. I set my mind that I'm gonna be a programmer someday and I worked hard for it. Of course, it wasn't a pleasant course, rather an exhausting, stress-inducing, and fearful ride. There were so many times I wanted to just quit, but I am glad I didn't. The road is long and I still have so much to do, but I'm happy I disciplined myself and chose this path.

Where I'm headed

I am at an early stage in my career so I'm keen on gaining experience in all fields of software development, however, I've grown to love front-end development. Something is satisfying about being able to make changes to code and instantly seeing it reflected on a web page for everyone to see. I want to work on bigger projects, besides intelligent and creative people, to learn, to create, to develop.
